# HTR-PM Interactive

A TypeScript web application for interactive analysis of a one-dimensional thermal-hydraulic model of the HTR-PM reactor.

The app is based on a model originally developed in an `ipynb` notebook and later adapted, with AI assistance, into an interactive web interface. The underlying model was prepared as part of an academic project at AGH University of Krakow, in the Nuclear Power Engineering program, for the Nuclear Reactor Thermal-Hydraulics course.

It makes it easier to adjust input parameters, inspect plots, compare operating scenarios, and check acceptance limits and KTA correlation ranges without manually recalculating the notebook.

## Language

The application interface is currently available in Polish only. English language support has not been added yet.
